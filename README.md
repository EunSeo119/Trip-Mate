## 프로젝트 규칙 
### `커밋 메세지 컨벤션`

- `Feature`: 새로운 기능
- `BugFix`: 버그 수정
- `Docs`: README, wiki 문서 수정
- `Style`: 코드 변경 없이 스타일 변경
- `Refactor`: 리팩토링
- `Chore`: 동작에 영향이 없는 코드 변경 없는 변경사항(주석 추가 등등)
- `Update`: 변경사항
